# FFA (Final Fantasy Adventure) Remake

2000年代前半に流行したCGIブラウザゲーム「FFA」を現代風にリメイクしたプロジェクトです。
HTML/CSS/JavaScriptのみで構成されており、ブラウザだけでサクサク遊べます。

## 🌟 特徴
- **レトロ×モダン**: 当時の殺伐としたテキストログの良さを残しつつ、プレミアムなダークモードUIとアニメーションを融合。
- **成長システム**: レベルアップ、武器の購入、そして強力な上位職への転職。
- **エリア移動**: 平原から始まり、森、そして無限地獄へと続く冒険。
- **永続保存**: `localStorage` を使用し、ブラウザを閉じてもデータが維持されます。

## 🕹️ 遊び方
1.  このリポジトリをクローンするか、GitHub Pagesで公開されたURLにアクセスします。
2.  「モンスターを探す」を連打して経験値とゴールドを稼ぎます。
3.  ゴールドが貯まったら「武器屋」で装備を強化。
4.  Lv10, 30, 60で「転職所」に行き、さらなる高みを目指しましょう。

## 🛠️ 技術スタック
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla ES6+)
- GitHub Pages (Hosting)
